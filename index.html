Para funcionar no seu navegador, o app precisa de uma chave de API do Google Gemini. É gratuita e leva 2 minutos para conseguir.
Como conseguir sua chave
1. Acesse aistudio.google.com
2. Faça login com sua conta Google
3. Clique em "Get API Key" no menu lateral
4. Clique em "Create API key"
5. Copie o texto que aparecer e cole abaixo
A chave fica salva apenas no seu navegador, neste dispositivo. O app não a envia para nenhum outro lugar além do Google.
